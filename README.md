# Placeholder Name CV

This repository hosts a placeholder CV template as a static GitHub Pages site. All names,
schools, and employers are generic placeholders.

## Live Site

- Production URL: `https://cv-template-example.shoug-tech.com`

## Project Structure

- `index.html`: the complete CV website, including styles and JavaScript
- `.github/workflows/static.yml`: GitHub Actions workflow that deploys the site to GitHub Pages
- `.gitignore`: local git ignore rules

## Features

- Bilingual layout with English and Arabic toggle
- Light and dark theme toggle
- Print to PDF support
- Download current CV as HTML
- Single-file deployment with no build step

## Deployment

The site is deployed automatically by GitHub Actions when changes are pushed to the `main` branch.

## Editing

To update the CV:

1. Edit `index.html`
2. Commit the changes
3. Push to `main`

GitHub Pages will rebuild the site automatically.
